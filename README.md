# 📊 Statistical Analysis & Linear Regression Project

This repository contains a Jupyter Notebook implementing key concepts from **statistics and machine learning**, including confidence intervals, data analysis, and linear regression using gradient descent.

---

## 🚀 Project Overview

This project demonstrates a combination of **statistical inference** and **machine learning techniques** applied to real-world-style datasets. The implementation is done from scratch wherever possible to strengthen conceptual understanding.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-learn (for preprocessing only)
- SciPy

## ✨ Key Implementations

---

### 🔹 1. Statistical Analysis (Confidence Intervals)

- Estimated population mean using sample data
- Calculated:
  - Standard Error (SE)
  - Margin of Error
  - Confidence Intervals (95%)
- Used:
  - `scipy.stats`
  - `statistics` module

#### Problems Solved:
- Average screen time estimation
- Proportion of public transport users

---

### 🔹 2. Data Analysis using Pandas

- Loaded and analyzed **Life Expectancy dataset**
- Computed:
  - Mean
  - Standard deviation
  - Sample size
  - Standard Error
- Constructed confidence intervals using real dataset

---

### 🔹 3. Linear Regression (From Scratch)

Implemented a complete linear regression pipeline without using built-in models:

#### 📌 Steps:
- Data loading and preprocessing
- Train-test split (75% / 25%)
- Feature normalization using `StandardScaler`
- Scatter plot visualization

#### 📌 Core Implementations:
- Prediction function  
- Cost function (Mean Squared Error)  
- Gradient Descent algorithm  

#### 📌 Model Training:
- Iterative optimization using gradient descent
- Learning rate tuning
- Cost tracking over iterations

#### 📌 Output:
- Final model coefficients
- Regression line visualization
